Hello there, let me take a moment to explain what's going on here...

This project is an SMALL implementation (supported commands) of the Logo language for turtle graphics. The help page will provide help with the supported syntax and semantics. If you would like more information Logo please see the wikipedia page.

 

This project was written by @SimpsonPatrick using JavaScript, jQuery, and Paper.js

NOTE: This project will NOT work in IE 8 and below. If you would like to use this application, please use a modern browser. (Chrome, Firefox, Safari, IE9+ etc.)

The code is Open Source: https://github.com/patricksimpson/weblogo

Feel free to submit issues, bugs, comments, on github. Tweet me, or you can email me your cool logo app at weblogoapp at gmail dot com.

If it is really cool, I will put up a page with user submitted logo programs / screenshots page!
