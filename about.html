About

Welcome to the home page for the WebLogo project.

Let me take a moment to explain what's going on here...

This project is an small implementation of the Logo language for turtle graphics. The help page will provide help with the supported syntax and semantics. If you would like more information Logo please see the wiki page.

 

This project was produced by Patrick Simpson using JavaScript, HTML5 (Canvas), jQuery, and Paper.js

NOTE: This project will not work in the current version of IE (9) and below. If you would like to use this web application, please use a modern standards based browser. (Chrome, Firefox, Safari, etc.)

Produced in 2012 by Patrick Simpson for Franklin University CS Capstone. There is no Copyright.
